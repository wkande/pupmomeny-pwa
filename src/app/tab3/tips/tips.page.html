Wallets

We create a new wallet for your account when you first use the app called Default Wallet. Your Default Wallet can be renamed but not deleted. You can create as many wallets as you link. Most users only have one but sometimes you might want to track expenses separately for other reasons.

The blue bar accross the bottom shows which is the current wallet, or the wallet you are working with.

Add-Edit-Delete Wallet

Add a new wallet by selecting the Add button.

Update a wallet by clicking on it or swiping it to the right. Swipe to the right to delete the wallet.

Switching between wallets is easy, just click on the Switch button and pick the desired wallet.

Categories

All wallets start out with a set of catgeories that we have added. Your expenses are grouped into catgories. You can use these catories as is or change the list however you like.

Add-Edit-Delete Category

Add a new category by selecting the Add Category button at the bottom of the Expenses tab.

Change the name of a category or delete it by swiping it to the right.

Vendors

Every time you enter an expense you will pick a category for it. You then have the option of entering or picking a vendor name for the expense. Each category has its own list of vendors. This keeps the vendors list short and only relavent to the catgegory.

Add-Edit-Delete Vendor

When entering a new expense click on the vendors button to open the vendor pick list. Then click on hte Manage List button at the bottom.

Add a new vendor by selecting the Add Line button, then enter a new vendor name.

Change the name of a vendor simply by editing a line.

Use the X button next to a vendor name to remove it..

Don't forget to click the Save button to save everything.
